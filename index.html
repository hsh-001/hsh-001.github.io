1-7

段寄存器

  1. 8060有四个段寄存器：CS,DS,SS,ES
  2. CS为代码段寄存器，IP为指令寄存器。
  3. jump 某一合法寄存器
  4. jump ax （类似与move IP，ax)

简单指令

  1. move
     mov al，[0]  将数据直接送入寄存器  将一个寄存器中的内容送入另一个寄存器
 mov 寄存器， 数据      mov ax,8
 mov 寄存器， 寄存器    mov ax,bx
 mov 寄存器， 内存单元  mov ax,[0]
 mov 内存单元，寄存器   mov [0],ax
 mov 段寄存器，寄存器   mov ds,ax
 mov 寄存器， 段寄存器
  2. add
     add 寄存器， 数据      add ax,8
 add 寄存器， 寄存器    add ax,bx
 add 寄存器， 内存单元  add ax,[0]
 add 内存单元， 寄存器  add [0],ax
  3. sub
     sub 寄存器， 数据      sub x,9
 sub 寄存器， 寄存器    sub ax,bx
 sub 寄存器， 内存单元  sub ax,[0]
 sub 内存单元， 寄存器  sub [0],ax

栈

  1. SS:存放栈顶的段地址; SP:存放栈顶的偏移地址;
  2. pop ax 将SS:SP送入ax，SP=SP+2
  3. push ax SP=SP-2
  4. push ds 将一个段寄存器的数据入栈
  5. pop es 出栈，用一个段寄存器接受出栈的数据
  6. push [0] 将一个内存单元的字入栈
  7. pop [0] 出栈，用一个内存字单元接收出栈的数据

1.5

从今天开始，学习汇编语言。

基础知识

  1. 寄存器：cpu可以存储数据的元件，一个cpu有多个寄存器
  2. 汇编语言组成：
     - 汇编指令（机器码的助记符）（汇编语言的核心，决定汇编语言的特性）
 - 伪指令（由编译器执行）
 - 其他符号（由编译器识别）
  3. 存储器：
     - cpu是计算器的核心部件
 - 存储器被划分为若干个存储单元 每个存储单元从0开始编号
  4. cpu对数据的读写，进行三类信息的交互：
     - 存储单元的地址（地址信息）
 - 器件的选择,读或写命令（控制信息）
 - 读或写的数据（数据信息）
  5. cpu对外部器件的控制是通过控制总线来进行的。控制总线是一些不同控制线的集合,有多少控制总线，就意味着cpu提供了对外部件的多少种控制。
  6. 一个cpu由运算器、控制器、寄存器等器件组成，这些组件靠内部总线相连。
     - 内部总线 cpu内部
 - 外部总线 cpu和主板上其他器件。
  7. 8060cpu有14个寄存器
     - AX,BX,CX,DX,SI,DI,SP,BP,IP,CS,SS,DS,ES,PSW
 - AX可以分为AH和AL（前者填0)均可独立使用
 - BX,CX,DX同理
  8. 16位结构cpu：
     - 运算器一次最多可以处理16位数据
 - 寄存器的最大宽度为16位
 - 寄存器和运算器之间的通路是16位的
  9. 基础地址+偏移地址=物理地址
  10. 段地址*16+偏移地址=物理地址
  11. 内存没有分段，段的划分来自cpu

    总结

    今天是开始的第一天，只学了一些基础的概念
