段寄存器

  1. 8060有四个段寄存器：CS,DS,SS,ES
  2. CS为代码段寄存器，IP为指令寄存器。
  3. jump 某一合法寄存器
  4. jump ax （类似与move IP，ax)

简单指令

  1. move
     mov al，[0]  将数据直接送入寄存器  将一个寄存器中的内容送入另一个寄存器
 mov 寄存器， 数据      mov ax,8
 mov 寄存器， 寄存器    mov ax,bx
 mov 寄存器， 内存单元  mov ax,[0]
 mov 内存单元，寄存器   mov [0],ax
 mov 段寄存器，寄存器   mov ds,ax
 mov 寄存器， 段寄存器
  2. add
     add 寄存器， 数据      add ax,8
 add 寄存器， 寄存器    add ax,bx
 add 寄存器， 内存单元  add ax,[0]
 add 内存单元， 寄存器  add [0],ax
  3. sub
     sub 寄存器， 数据      sub x,9
 sub 寄存器， 寄存器    sub ax,bx
 sub 寄存器， 内存单元  sub ax,[0]
 sub 内存单元， 寄存器  sub [0],ax

栈

  1. SS:存放栈顶的段地址; SP:存放栈顶的偏移地址;
  2. pop ax 将SS:SP送入ax，SP=SP+2
  3. push ax SP=SP-2
  4. push ds 将一个段寄存器的数据入栈
  5. pop es 出栈，用一个段寄存器接受出栈的数据
  6. push [0] 将一个内存单元的字入栈
  7. pop [0] 出栈，用一个内存字单元接收出栈的数据
